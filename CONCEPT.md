# MindfulMoments - Концепция приложения для медитации

## 📱 Обзор
**MindfulMoments** - мобильное Android приложение для медитации и снятия стресса, работающее полностью оффлайн.

## ✨ Ключевые особенности

### Функциональность
- 🧘 **Практики медитации**: дыхательные упражнения, body scan, визуализация
- 🫁 **Тренажер дыхания**: визуальный гид с анимациями
- ⏱️ **Таймер медитации**: настраиваемые сессии с звуковыми сигналами
- 📊 **Статистика**: отслеживание прогресса и достижений
- 🌙 **Режимы**: светлая и темная темы
- 📴 **Оффлайн**: полная работа без интернета

### Практики для снятия стресса

#### 1. Дыхательные техники
- **4-7-8 дыхание**: вдох 4 сек, задержка 7 сек, выдох 8 сек
- **Box breathing**: вдох-задержка-выдох-задержка по 4 секунды
- **Глубокое дыхание**: медленное диафрагмальное дыхание
- **Энергизирующее дыхание**: быстрое дыхание для бодрости

#### 2. Медитативные практики
- **Body Scan**: сканирование тела от головы до ног
- **Mindfulness**: осознанность настоящего момента
- **Визуализация**: успокаивающие образы (океан, лес, горы)
- **Loving Kindness**: медитация любящей доброты

#### 3. Быстрые упражнения (1-5 минут)
- **5-4-3-2-1**: техника заземления через органы чувств
- **Мышечная релаксация**: напряжение и расслабление мышц
- **Наблюдение за мыслями**: техника "облака"

## 🧭 Навигация

### Структура экранов

```
┌─────────────────────────────┐
│     Home (Главный)          │
│  - Быстрый старт            │
│  - Рекомендованные практики │
│  - Прогресс дня             │
└─────────────────────────────┘
         ↓ Bottom Nav
┌─────────────────────────────┐
│   Practices (Практики)      │
│  - Дыхание                  │
│  - Медитация                │
│  - Быстрые техники          │
└─────────────────────────────┘
         ↓ Bottom Nav
┌─────────────────────────────┐
│     Timer (Таймер)          │
│  - Настройка длительности   │
│  - Выбор фона               │

         ↓ Bottom Nav
┌─────────────────────────────┐
```

### Bottom Navigation
- 🏠 Главная
- 🧘 Практики
- ⏱️ Таймер
- 📊 Прогресс

### UX правила
1. **Правило одной руки**: все важные элементы в зоне большого пальца
2. **Минимум кликов**: до любой практики не более 2 тапов
3. **Понятные подсказки**: при первом запуске онбординг
4. **Обратная связь**: визуальный и тактильный отклик на действия
5. **Жесты**: свайп для навигации между экранами
6. **Прогрессивное раскрытие**: показываем только нужное

## 🏗️ Техническая архитектура

### Технологии
- **Язык**: Kotlin
- **UI**: Jetpack Compose
- **Архитектура**: MVVM + Clean Architecture
- **БД**: Room (SQLite) для оффлайн хранения
- **DI**: Hilt
- **Асинхронность**: Kotlin Coroutines + Flow
- **Навигация**: Compose Navigation

### Структура проекта
```
app/
├── data/
│   ├── local/          # Room БД, DAO
│   ├── repository/     # Репозитории
│   └── model/          # Модели данных
├── domain/
│   ├── usecase/        # Бизнес-логика
│   └── model/          # Domain модели
└── presentation/
    ├── ui/             # Compose UI компоненты
    ├── viewmodel/      # ViewModels
    └── theme/          # Темы и стили
```

### Оффлайн возможности
- ✅ Все данные хранятся локально в Room
- ✅ Звуки встроены в APK
- ✅ Никаких сетевых запросов
- ✅ Работа без permissions
